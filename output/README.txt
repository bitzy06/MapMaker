This folder contains output files generated by MapMaker.

When you click "Export to Output", processed shapefiles will be saved here.
Files will be prefixed with the map type (country_ or state_) and suffixed with "_output".

Example outputs:
- country_output.shp
- country_output.dbf  
- country_output.shx
- state_output.shp
- state_output.dbf
- state_output.shx
